# Megaline Revenue Analysis

## Business Context
Megaline is a telecommunications operator that offers two prepaid plans: Surf and Ultimate. 
The marketing department wants to determine which plan generates higher average revenue in order to optimize advertising budget allocation.

## Objective
Analyze customer behavior and determine which prepaid plan generates more revenue on average using statistical hypothesis testing.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Dataset
500 customers from 2018 including:
- Calls
- SMS
- Internet usage
- Plan type
- Geographic information

## Methodology
1. Data cleaning and preprocessing
2. Monthly aggregation per user
3. Revenue calculation based on plan conditions
4. Exploratory Data Analysis (EDA)
5. Hypothesis testing (t-test with Levene test)

## Key Findings
- Ultimate users generate higher and more stable revenue.
- Surf users show more variability and frequent overage charges.
- Statistical testing confirms significant revenue differences between plans.
- No strong evidence of revenue difference between NY-NJ area and other regions.

## 📌 Conclusion
Ultimate provides stable and predictable revenue, while Surf generates more volatile but potentially higher overage-based income.
